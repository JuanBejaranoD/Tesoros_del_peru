# Tesoros del Perú

Un sitio web dedicado a promover la rica gastronomía y los impresionantes lugares históricos del Perú.

## Estructura
- **Inicio**: Visión general.
- **Recetas**: Platos emblemáticos como Ceviche y Lomo Saltado.
- **Lugares**: Destinos históricos como Machu Picchu y Cusco.

## Stack
- HTML5, TailwindCSS, React 18.
- Iconos: Lucide.
- Imágenes: Unsplash.
